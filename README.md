# Word Frequency

#### Independent project for practicing BDD with PHP for Epicodus, 02.17.2017

#### By Patrick McGreevy

## Description

This Silex website calculates how frequently a word appears in a given string.


## Setup/Installation Requirements
1. Set project root as working directory in CLI.
2. Run `$ composer install --prefer-source --no-interaction`.
3. Run `$ cd web`.
4. Run `$ php -S localhost:8000`.
5. Visit **`localhost:8000`** in web browser.
